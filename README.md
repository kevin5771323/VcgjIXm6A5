# 项目简介

## 核心功能
- 基于YOLOv8的葡萄叶病检测系统，支持4类病害识别：black measles、black rot、grapeleaf blight和健康叶片
- 提供图片、视频及摄像头三种检测方式
- 支持批量检测功能，可查看每一张检测结果
- 提供自定义置信度和IOU阈值设置
- 支持选择显示标签和原图
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持检测结果保存

## 适用场景
- 葡萄种植病害监测与管理
- 农业科研与教学
- 植物病理学研究
- 农业生产自动化检测

## 技术特点
- 使用Python3.9开发，依赖环境包括opencv-python、PyQt5、ultralytics、torch1.9等
- 提供完整数据集（1500张标注图片）
- 包含训练好的模型及训练结果
- 提供完整源码和UI界面源文件
- 包含项目运行教程和说明文档
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
